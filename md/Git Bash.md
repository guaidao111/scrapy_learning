# Git Bash



| COMMAND                    | USAGE                                             |
| -------------------------- | ------------------------------------------------- |
| git status                 | 查看目前是否全clear，文件tree是否clean            |
| git add filename.type      | 将tree中需要添加的文件送入staging area            |
| git commit (-m 'sjdjdsks') | 将staging中的文件送入本地database，可选-m传递信息 |
| git log                    | 查看之前进行commit的历史记录log信息               |
| ctrl+L                     | 清除控制台信息                                    |
| git difftool HEAD          | 查看和之前版本的区别                              |
|                            |                                                   |
